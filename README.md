# Website
The website was built using React.
How to start the website:
1. In the terminal:
   
    a. Install dependencies
   
        "npm run init"
   
    b. Open static website

       "npm run frontend"
   
   Alternate method:
   
            1. "cd features/frontend"
   
            2. "npm start"
   
As of right now, the website is only hosted locally
